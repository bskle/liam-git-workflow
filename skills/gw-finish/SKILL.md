# Finish Workflow

Read first:

- `../../references/policy.md`
- `../../references/pr-rules.md`
- `../../references/branch-matrix.md`

## Workflow

1. Identify the branch family
2. State the correct merge target
3. Apply the finish checklist
4. Suggest PR summary sections
5. Remind about branch cleanup after merge

## Output

Provide:

- merge target
- finish checklist
- PR summary outline
- cleanup reminder
