# Release Workflow

Read first:

- `../../references/policy.md`
- `../../references/pr-rules.md`

## Workflow

1. Confirm release intent
2. Check that integration has already happened on `dev`
3. State that release merge goes to `main`
4. Recommend semantic tag with `v` prefix
5. Remind about release notes

## Output

Provide:

- release branch or merge recommendation
- merge target
- tag recommendation
- release checklist
